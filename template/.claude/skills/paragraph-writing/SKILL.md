# パラグラフ・ライティング／学術的文章作法

レポート・論文を「論理的で・構造的で・分かりやすい」文章にするための作法。論文を書く・推敲する・構成を設計するときは、以下の原則で文章とアウトラインを点検する。出典は神戸大学総合図書館 LS「パラグラフ・ライティング講座」(2024-02-07)。

## 学術的文章とは

自分が立てた**問い**を、**客観的事実**で**論証**しながら、**答え**を明示する文章（感想文＝主観的な体験・心情の記述、とは異なる）。次の大構造を持ち、これに沿えば書ける。

| 要素 | 役割 | 例 |
| --- | --- | --- |
| 問い | 何を明らかにするか | ○○は本当にそうか? |
| 主張 | 問いへの答え（自分の立場） | 従来は○○とされてきたが、じつは□□である。 |
| 論証 | 主張を支える客観的事実・根拠 | 調査で△△という結果が得られたのが証左である。 |
| 結論 | 論証を踏まえた答えの確定 | ○○は、△△という調査から□□である。 |

## パラグラフの構成要素

1つのパラグラフは次から成る。

- **トピック・センテンス**: そのパラグラフで何を述べるかを**一文で**示したもの。パラグラフの「主張」そのもの。
- **サブ・センテンス**: トピック・センテンスを**展開する**文（言い換え・具体例・語句の説明・根拠）。
- **コンクルーディング・センテンス**（任意）: 最後に置くまとめの文。**あってもなくてもよい。**

例（4文構成 / 【】は各文の役割）:

> 【トピック】京都市内では、LUUPとよばれる電動マイクロモビリティが広く普及している。【サブ】LUUPとは、電動キックボード・電動自転車のシェアサービスのことである。スマホ一つあれば街中の電動自転車に簡単に乗り降りできる。【まとめ】その利便性等が評価され、京都市と連携しながら更なる設置が進められている。

## 二大原則

1. **トピック・センテンスは原則パラグラフの文頭に置く** — 何が書かれているか読み手に見通しを与える。
2. **一つのパラグラフの主張は一つに留める** — サブ・センテンスで主張と無関係なことを述べない。脱線した文は削るか別パラグラフへ。

目安: 隣り合うパラグラフは**接続詞**で論理的につなぐ。1パラグラフは**だいたい 200〜400 字**。

## 文章全体の階層構造とセクションの分け方

```
論文全体  … 問い → 主張 → 論証 → 結論 の大構造
└─ 章 └─ 節（セクション）  … 各セクションも「問い→主張→論証→結論」の小構造を持つ
      └─ サブセクション（必要なときだけ） └─ パラグラフ（最小単位） └─ 文
```

**サブセクションは、セクションで主張したいことが「並列で」複数あるときに分割するためのもの。**

- 主張が**並列に複数ある**（独立した論点が横に並ぶ）→ **サブセクションに分ける**。
- 主張が**順につながって展開する**（逐次的・直線的に積み上がる）→ サブセクションにせず、**一つのセクション内でパラグラフに分ければよい**。

逐次的な議論を無理にサブセクションへ切ると論理の流れが分断される。サブセクションは並列な論点を整理する道具と考える。

## トピック・センテンスの選び方

各パラグラフに主張（トピック・センテンス）を一つ立てる。**選定基準は「その主張がないと論文全体として成り立たないかどうか」。**

- その一文を削ると**論文全体の論理が崩れる**＝必須の主張＝トピック・センテンスに値する（独立パラグラフを与える）。
- なくても全体が成り立つ内容は、独立パラグラフに格上げせず、サブ・センテンス（具体例・補足）に留めるか削る。

この基準は点検項目「トピック・センテンスだけ読むとアウトラインになるか」と表裏一体。必須の主張だけが並べば、拾い読みで骨子が再構成できる。

## 執筆プロセス

1. まず**トピック・センテンスだけを書いて**全体の骨子（アウトライン）を整える。
2. 並べて読み、論理の流れ・抜け漏れ・並列／逐次の関係を点検する（セクション／サブセクション設計もここで決まる）。
3. その後、各トピック・センテンスを**サブ・センテンスで膨らませる**。

## 点検チェックリスト

パラグラフ単位:

- [ ] トピック・センテンスが**文頭**にあるか。

- [ ] サブ・センテンスがトピック・センテンスと**どう結びつくか説明できる**か（無関係な文がないか）。

- [ ] パラグラフの**トピックは一つ**に絞れているか。

パラグラフ構造（全体）:

- [ ] **トピック・センテンスだけ読むと論文のアウトライン（要約）になる**か。

- [ ] **効果的な接続詞**でパラグラフがつながっているか。

- [ ] 並列の論点はサブセクション、逐次の議論はパラグラフ分け、と振り分けられているか。
